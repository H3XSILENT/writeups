# Active Directory
O Active Directory da Microsoft é a espinha dorsal do mundo corporativo. Ele simplifica o gerenciamento de dispositivos e usuários em um ambiente corporativo. Nesta sala, vamos explorar em detalhes os componentes essenciais do Active Directory.

Literalmente, se você quiser ser um pentest corporativo você precisará de AD.

Não espero que seja um trópico simples (e não sera).
